android
----------
cd android

a) development version
./build.sh  
./push.sh

b) deploy version
./build.sh deploy

both versions:
./start.sh

there are offline and online versions: change the flag in ACProjectView constructor
for offline development version you should use ./pushoffline.sh instead of ./push.sh
